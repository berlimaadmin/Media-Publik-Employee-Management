# Media Publik Employee Management System

A web-based management system for Media Publik to manage:

## Features

- Employee Data Management
- Attendance Management
- Work Schedule Management
- Project List Management
- Project Reporting

This system is designed to centralize employee and project management processes, improve operational efficiency, and provide accurate reporting for management.
